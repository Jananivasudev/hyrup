Screen 1

🏠 Home Dashboard

The first screen users see. A personalized hub — not just a feed.

  • Greeting "Hi Rahul! 👋" — warm, personal. Mode toggle + bell on right
  • Daily Challenge Full-width hero card (orange gradient). Shows prompt + 🔥 streak counter. CTA: "+ Create Post"
  • Week Activity 7-day row (Mon–Sun). Filled = active, outlined = inactive. Today = orange accent
  • Quick Cards 2×2 grid: New Connections, Active Groups, Trending Win, Study Group. Tap → navigates to tab
  • Feed Preview 2 trending posts in compact list. "See All →" pulls user into Feed
Screen 2

📰 Feed — Full-Image Cards

Image fills the entire card. User info at top, content at bottom. 2 stacked shadows behind.

Position Content
Top-left User avatar (circle) + Name + College — white text on dark gradient
Top-right Post type: "🏆 Win" / "🚀 Project" / "💡 Thought" — glass pill
Bottom Post text (bold 20px) + ❤️ 142 · 💬 28 · 📤 · 🔖
Behind 2 stacked shadow cards at 95% and 90% scale for depth
Gesture Action
← Swipe left Next post
→ Swipe right Bookmark + toast "🔖 Post Saved!"
Tap card Expand → comments view
Screen 3

🤝 Connect — Full-Photo Cards

Profile photo fills the entire card. Info overlaid in white. Like a dating app — but for networking.

Position Content
Top-left "Why" badge: "Same domain 🎨" / "Same city 📍" — frosted glass pill
Top-right Level: "Lv 8" — green glass pill
Bottom 📍 College → Name ✦ (26px bold) → Skill pills: "🔹 UI/UX" "🔹 Figma"
Gesture Action
→ Swipe right Send connection request
← Swipe left Skip to next person
Buttons ✗ (red) / ✓ (green) — fallback below card

Also: Requests tab (accept/decline list) · Nearby tab (radius filter + distance)

Screens 4 & 5

💬 Chat · 👤 Profile

💬

Unified Chat

DMs + Groups in one inbox. Filter tabs: All / DMs / Groups. Groups have emoji badges (⚡📚🎮). Search bar at top. "+" to create.

👤

Profile

Gradient cover + avatar. Stats: Connections · Posts · Skills · Wins. Verified skill pills. "✦ AI Resume" button. Tabs: Posts grid / AI Highlights / Bookmarks.
